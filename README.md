# 🚀 Git шпаргалка (быстрый пуш)

## 📤 Отправить изменения на GitHub

### Полный вариант

```bash

git add README.md

git commit -m "описание изменений"

git push origin master

-------------------------------------
**Пояснения.**

git add .	                    - Добавить все изменённые файлы

git add README.md	            - Добавить только README

git commit -m "текст описания"	- Сохранить изменения с описанием

git push origin master	        - Отправить на GitHub

-------------------------------------

**Мой рабочий цикл**

0. Внести изменения в файлы, тесты, readme.

1. Изменить файл → Ctrl + S

2. git add .

3. git commit -m "что сделал"

4. git push

-------------------------------------
